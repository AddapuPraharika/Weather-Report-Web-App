# Weather-Report-Web-App
This is a simple web application that provides real-time weather information for any city you choose. It displays the current weather conditions, including temperature, humidity, wind speed, and a brief description.
GitHub

🔍 Features
Current Weather: Displays the current temperature, humidity, wind speed, and weather description.

Location Search: Enter the name of any city to get its current weather.

User-Friendly Interface: Clean and intuitive design for easy navigation.

View the Weather:
